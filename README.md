# The Midnight Line — Vertical Slice

This is a minimal Vite + TypeScript vertical slice for a narrative adventure.

Run:

1. Install dependencies:
```
npm install
```
2. Start dev server:
```
npm run dev
```

Open the URL printed by Vite (usually http://localhost:5173) and play.

Controls:
- Move: Arrow Left / Arrow Right or A / D
- Interact: Enter or Space or click on interactables
- Choose dialog options: click an option or press 1/2/3
