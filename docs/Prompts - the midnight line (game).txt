You are an expert TypeScript/React game developer.
Build the first vertical slice of a browser-based narrative game called “The Midnight Suburban Line”.
High-level vision

* Style: clean, minimal UI (like a modern interactive fiction reader), with subtle atmospheric touches (dark theme, hints of train/suburban imagery), but no heavy graphics needed yet.
* Core loop for this slice:
   1. Player sees a title / start screen.
   2. Player chooses who this night’s train is secretly for (from a few archetypes).

  - reads a short scene,
